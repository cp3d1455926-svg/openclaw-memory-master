# Email Writer - 邮件写作助手

📧 智能邮件写作助手，快速生成专业邮件！

## 功能

- 📝 邮件模板生成
- ✉️ 多场景支持
- 🌍 中英文切换
- 🎯 语气调整
- 📋 自动格式化
- ✅ 语法检查
- 🔔 邮件跟进提醒
- 📊 邮件效果分析

## 使用方法

### 生成邮件
```
写邮件 请假
写邮件 会议邀请
写邮件 项目汇报
```

### 指定场景
```
邮件模板 商务合作
邮件模板 客户跟进
邮件模板 投诉建议
```

### 调整语气
```
写邮件 催款 语气：正式
写邮件 感谢 语气：亲切
```

### 语言切换
```
写邮件 合作 英文
write email cooperation Chinese
```

## 特性

✅ 50+ 种邮件模板
✅ 中英文双语支持
✅ 正式/非正式语气
✅ 自动添加称呼/落款
✅ 邮件主题优化
✅ 附件提醒
✅ 抄送建议
✅ 发送时间建议

## 配置

在 `config.json` 中设置：
- `defaultTone`: 默认语气
- `defaultLanguage`: 默认语言
- `signature`: 默认签名
- `templates`: 自定义模板
