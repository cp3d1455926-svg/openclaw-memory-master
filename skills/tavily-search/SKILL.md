# Tavily Search - 智能搜索助手

🔍 基于 Tavily API 的智能搜索工具，获取高质量搜索结果！

## 功能

- 🔍 智能网页搜索
- 📰 新闻资讯搜索
- 📚 学术研究搜索
- 💻 技术文档搜索
- 🌍 多语言支持
- 📊 结果去重排序
- 🎯 深度搜索模式
- 📝 摘要生成

## 使用方法

### 基础搜索
```
搜索 关键词
search keyword
 Tavily 搜索 人工智能
```

### 高级搜索
```
深度搜索 机器学习 2026
新闻搜索 AI 大模型
学术搜索 深度学习
```

### 指定参数
```
搜索 Python 教程 数量：10
搜索 天气预报 天数：7
```

## 配置

在 `config.json` 中设置：
- `api_key`: Tavily API 密钥
- `search_depth`: 搜索深度 (basic/advanced)
- `max_results`: 最大结果数
- `include_images`: 包含图片
