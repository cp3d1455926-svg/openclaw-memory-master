# 📖 Flashcard - 单词卡片

基于艾宾浩斯记忆曲线的单词学习工具。

## 功能列表

- ✅ 每日单词推送
- ✅ 记忆曲线复习
- ✅ 学习进度追踪
- ✅ 单词测验
- ✅ 自定义词库

## 使用示例

```
今日单词：today
复习单词：review 1 1
随机单词：random 10
添加单词：add hello /həˈloʊ/ 你好
查看统计：stats
```

## 记忆等级

| 等级 | 说明 | 复习间隔 |
|------|------|---------|
| Lv.0 | 未学习 | - |
| Lv.1 | 生疏 | 1 天 |
| Lv.2 | 熟悉 | 2 天 |
| Lv.3 | 掌握 | 4 天 |
| Lv.4 | 熟练 | 7 天 |
| Lv.5 | 已掌握 | 15 天 |
