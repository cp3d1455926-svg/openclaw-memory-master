# Countdown Timer - 倒计时工具

⏰ 多功能倒计时，让时间更有仪式感！

## 功能

- ⏰ 倒计时设置
- 📅 重要日期提醒
- 🎯 目标追踪
- 📊 时间统计
- 🔔 多重提醒
- 🎨 自定义主题
- 📱 桌面小组件
- 📤 分享倒计时

## 使用方法

### 创建倒计时
```
倒计时 距离春节
倒计时 距离生日 2024-12-25
倒计时 距离考试 30 天
```

### 设置提醒
```
倒计时 会议 14:00 提醒
倒计时 约会 明天 10 点 提前 1 小时提醒
```

### 目标追踪
```
开始倒计时 减肥 目标：60kg
开始倒计时 学习 100 天
```

### 查看统计
```
查看倒计时
时间统计
倒计时记录
```

## 特性

✅ 多个倒计时同时
✅ 自定义提醒方式
✅ 精美界面主题
✅ 时间进度条
✅ 分享卡片生成

## 配置

在 `config.json` 中设置：
- `defaultReminder`: 默认提醒时间
- `theme`: 默认主题
- `soundEnabled`: 声音开关
- `shareFormat`: 分享格式
