# Book Summary - 书籍摘要生成

📚 快速生成书籍摘要，提取核心知识点！

## 功能

- 生成书籍核心摘要
- 提取关键知识点
- 制作思维导图大纲
- 生成读书笔记模板
- 支持多种书籍类型
- 推荐相关书籍

## 使用方法

### 生成摘要
```
书籍摘要 [书名]
读书笔记 [书名]
```

### 获取知识点
```
知识点 [书名] [章节]
金句 [书名]
```

### 获取推荐
```
推荐书籍 [类型/主题]
类似书籍 [书名]
```

## 支持类型

- 📖 商业经管
- 🔬 科学技术

- 🧠 心理学
- 💰 理财投资

## 示例输出

```
📚 《原子习惯》摘要

📖 作者：James Clear
⭐ 评分：4.8/5
📄 类型：自我提升

1️⃣ 微小习惯的力量

   - 提示 → 渴求 → 反应 → 奖励

3️⃣ 身份认同

   - 不是"我要跑步"，而是"我是跑步者"

□ 选择一个想养成的习惯

□ 设置环境提示
```
