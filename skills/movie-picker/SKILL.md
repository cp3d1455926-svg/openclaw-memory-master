# Movie Picker - 电影选择器

🎬 智能电影推荐，治愈选择困难症！

## 功能

- 🎥 随机电影推荐
- 🎭 按类型筛选
- ⭐ 豆瓣评分对接
- 🍿 观影记录管理
- 📅 观影计划
- 👥 多人投票选择
- 🎲 随机抽选
- 📝 影评生成

## 使用方法

### 随机推荐
```
推荐电影
随机电影
今天看什么电影
```

### 按类型
```
推荐科幻电影
推荐喜剧片
推荐高分悬疑片
```

### 根据心情
```
推荐电影 心情：开心
推荐电影 心情：难过
推荐电影 心情：无聊
```

### 多人选择
```
电影投票 星际穿越 流浪地球 三体
开始投票
```

## 配置

在 `config.json` 中设置：
- `defaultType`: 默认类型
- `minScore`: 最低评分
- `platform`: 首选平台
- `historyEnabled`: 观影记录
