# 🔌 API Tester - API 测试工具

RESTful API 快速测试、请求构建、响应分析。

## 功能列表

- ✅ GET/POST请求
- ✅ 自定义请求头
- ✅ JSON 数据处理
- ✅ 响应时间统计
- ✅ 内置测试模板

## 使用示例

```
GET 请求：get https://httpbin.org/get
POST 请求：post https://httpbin.org/post {"name":"test"}
测试 API: test httpbin
查看模板：templates
```

## 内置模板

- httpbin (HTTP 测试服务)
- jsonplaceholder (REST API 测试)
