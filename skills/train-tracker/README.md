2. 点击 **Settings**（设置）标签

1. 在左侧菜单找到 **Pages**
2. 点击进入 Pages 设置

### 步骤 3：配置 Pages
1. **Source** 选择：`Deploy from a branch`
2. **Branch** 选择：`master` 
3. **Folder** 选择：`/ (root)`
4. 点击 **Save**

## 💡 使用提示

1. **提前到达** - 建议提前 5-10 分钟到观赏点
2. **开启通知** - 列车到达前会提醒（需要浏览器权限）
3. **准备相机** - 长焦镜头 200mm+ 效果最佳
4. **最佳时间** - 傍晚 17-19 点光线最美
