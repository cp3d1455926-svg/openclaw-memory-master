# Invoice Generator - 发票/收据生成器

🧾 快速生成专业发票和收据，支持多种格式！

## 功能

- 🧾 发票模板生成
- 📄 收据/账单生成
- 💰 自动计算金额
- 📊 税率计算
- 🏢 公司信息管理
- 📱 电子发票支持
- 📤 批量生成
- 📧 邮件发送

## 使用方法

### 生成发票
```
生成发票 金额：5000 项目：设计服务
发票 客户：XX 公司 金额：10000
```

### 生成收据
```
生成收据 收到 XX 公司 定金 3000 元
收据 项目：咨询服务 金额：2000
```

### 管理信息
```
添加公司信息
添加客户信息 XX 公司
查看发票记录
```

### 批量操作
```
批量生成发票 客户列表.xlsx
导出本月发票
```

## 配置

在 `config.json` 中设置：
- `companyInfo`: 公司信息
- `taxRate`: 默认税率
- `invoicePrefix`: 发票编号前缀
- `currency`: 默认货币
