# Quiz Maker - 测验生成器

📝 自动生成知识测验，检验学习效果！

## 功能

- 根据主题生成测验题
- 多种题型（选择/判断/填空）
- 自动评分和解析
- 难度等级选择
- 错题本功能
- 学习报告生成

## 使用方法

### 创建测验
```
创建测验 [主题]
测验 [主题] [难度]
```

### 题型选择
```
选择题 [主题]
判断题 [主题]
填空题 [主题]
混合测验 [主题]
```

### 学习功能
```
错题本
复习错题
学习报告
```

## 难度等级

- 🟢 简单 - 基础知识点
- 🟡 中等 - 理解与应用
- 🔴 困难 - 综合与分析
- 🟣 专家 - 深度与拓展

## 文件结构

```
quiz-maker/
├── SKILL.md          # 技能说明
├── skill.json        # 技能配置
├── main.js           # 主程序
├── question-bank/    # 题库
│   ├── tech.json     # 科技类
│   ├── history.json  # 历史类
│   └── general.json  # 综合类
└── user-data/        # 用户数据
    └── progress.json # 学习进度
```
