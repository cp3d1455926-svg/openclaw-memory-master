# 📖 Explainer - 概念解释

复杂概念简单化讲解、类比说明、示例演示。

## 功能列表

- ✅ 简单解释
- ✅ 类比说明
- ✅ 实际例子
- ✅ 关键词提取
- ✅ 搜索功能

## 使用示例

```
解释概念：explain 区块链
完整解释：explain 区块链 full
搜索概念：search 数据
随机学习：random
```

## 内置概念

- 区块链
- 人工智能
- 云计算
- API
- 大数据

## 解释级别

- simple - 简单解释
- analogy - 类比说明
- example - 实际例子
- full - 完整解释
