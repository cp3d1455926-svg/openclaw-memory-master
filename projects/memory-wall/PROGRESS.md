## 💡 创意亮点

### 1. 情绪模式系统

受电影启发，设计了 4 种情绪模式：

- **乐乐模式** 🟡: 只显示高重要性记忆，金色温暖色调
- **忧忧模式** 🔵: 显示所有记忆，蓝色冷静色调
- **焦焦模式** 🟠: 显示待办事项，橙色警示色调
- **普通模式** 😐: 标准显示

### 2. 记忆球视觉编码

- **颜色**: 4 类记忆对应 4 种颜色
- **大小**: 重要性决定尺寸
- **亮度**: 访问次数影响发光强度
- **动画**: 核心记忆脉动，新记忆闪烁

### 3. 时间树记忆架

- **今天**: 新鲜记忆，完全展开
- **本周**: 近期记忆，可展开
- **本月**: 历史记忆，默认收起
- **更早**: 归档记忆

## 🚀 项目意义

Memory Wall 不仅是一个可视化界面，更是：

1. **AI 记忆的具象化**: 让抽象的记忆概念变得可见可感
2. **情感化设计**: 将电影元素融入工具，增加使用乐趣
3. **技术探索**: 实践 React + Three.js + 动画的最佳组合
4. **OpenClaw 生态**: 丰富 Memory-Master 的功能矩阵
