# Learning Path - 学习路径规划

🗺️ 为你定制系统化学习路径，从入门到精通！

## 功能

- 生成系统学习路径
- 分阶段学习目标
- 推荐学习资源
- 进度追踪
- 时间规划建议
- 技能图谱可视化

## 使用方法

### 创建路径
```
学习路径 [技能/主题]
规划 [技能] [时间]
```

### 查看进度
```
我的进度
今日任务
本周目标
```

### 获取资源
```
学习资源 [主题]
推荐教程 [技能]
```

## 支持领域

- 💻 编程语言（Python/JS/Java 等）
- 🤖 人工智能/机器学习
- 🌐 Web 开发
- 📊 数据分析
- 🔒 网络安全
- 📱 移动开发
- 🎨 设计/UIUX
- 📈 产品运营
- 💼 职场技能

## 文件结构

```
learning-path/
├── SKILL.md          # 技能说明
├── skill.json        # 技能配置
├── main.js           # 主程序
├── paths/            # 学习路径模板
│   ├── python.json
│   ├── web-dev.json
│   └── ml.json
└── user-progress/    # 用户进度
    └── progress.json
```
