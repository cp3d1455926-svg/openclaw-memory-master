# 🛒 Shopping List - 购物清单

智能分类、比价提醒、预算管理的购物助手。

## 功能列表

- ✅ 添加/删除购物项
- ✅ 智能分类
- ✅ 优先级管理
- ✅ 预算管理
- ✅ 购买统计
- ✅ 清空已完成

## 使用示例

```
添加商品：add 苹果 #水果 5 10
添加商品：add 牛奶 #乳品 2 15 high
查看清单：list
标记购买：buy 1 9.5
删除商品：remove 1
清空已完成：done
查看统计：stats
```

## 商品分类

| 分类 | 示例 |
|------|------|
| 🥬 生鲜 | 蔬菜、水果、肉类、海鲜 |
| 🥛 乳品 | 牛奶、酸奶、奶酪 |
| 🍞 主食 | 面包、米饭、面条 |
| 🥤 饮料 | 水、果汁、茶、咖啡 |
| 🍪 零食 | 饼干、薯片、坚果 |
| 🧴 日用 | 洗发水、纸巾、洗衣液 |
| 💊 药品 | 感冒药、创可贴、口罩 |

## 优先级

- 🔴 high - 高优先级（必买）
- 🟡 normal - 中优先级（常规）
- 🟢 low - 低优先级（可选）
