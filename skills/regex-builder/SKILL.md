# 🔤 Regex Builder - 正则生成器

可视化正则表达式、常用模式、在线测试。

## 功能列表

- ✅ 常用模式库
- ✅ 正则测试
- ✅ 语法速查
- ✅ 模式解释
- ✅ 构建向导

## 使用示例

```
查看模式：patterns
测试正则：test \d+ abc123
匹配邮箱：match 邮箱 test@example.com
匹配手机：match 手机号 13800138000
查看语法：syntax
```

## 预定义模式

| 模式 | 说明 |
|------|------|
| 邮箱 | 电子邮件地址 |
| 手机号 | 中国手机号 |
| URL | 网址 |
| IP 地址 | IPv4 地址 |
