# 📚 学习类 Skills 合集

小鬼创建的学习类 OpenClaw Skills，帮助你更高效地学习！

---

## 📖 已创建技能

### 1️⃣ Vocab Builder - 每日单词
**位置**: `vocab-builder/`

📚 每日学习新单词，提升词汇量！

**功能**:
- ✅ 每日推送新单词（支持英语、日语等）
- ✅ 单词详情：释义、例句、发音
- ✅ 单词卡片复习
- ✅ 学习进度追踪
- ✅ 支持自定义词库

**快速开始**:
```
每日单词 英语
查单词 apple
复习单词
测试我
```

---

### 2️⃣ Code Snippet - 代码片段管理
**位置**: `code-snippet/`

💻 管理和分享你的代码片段，建立个人代码库！

**功能**:
- ✅ 保存常用代码片段
- ✅ 按语言/标签分类
- ✅ 快速搜索和检索
- ✅ 代码高亮显示
- ✅ 支持多种编程语言
- ✅ 一键复制代码

**快速开始**:
```
保存代码 [语言] [标签]
搜索代码 [关键词]
我的代码 [语言]
```

---

### 3️⃣ Book Summary - 书籍摘要生成
**位置**: `book-summary/`

📚 快速生成书籍摘要，提取核心知识点！

**功能**:
- ✅ 生成书籍核心摘要
- ✅ 提取关键知识点
- ✅ 制作思维导图大纲
- ✅ 生成读书笔记模板
- ✅ 支持多种书籍类型
- ✅ 推荐相关书籍

**快速开始**:
```
书籍摘要 [书名]
读书笔记 [书名]
知识点 [书名] [章节]
推荐书籍 [类型]
```

---

### 4️⃣ Quiz Maker - 测验生成器
**位置**: `quiz-maker/`

📝 自动生成知识测验，检验学习效果！

**功能**:
- ✅ 根据主题生成测验题
- ✅ 多种题型（选择/判断/填空）
- ✅ 自动评分和解析
- ✅ 难度等级选择
- ✅ 错题本功能
- ✅ 学习报告生成

**快速开始**:
```
创建测验 [主题]
选择题 [主题]
错题本
学习报告
```

---

### 5️⃣ Learning Path - 学习路径规划
**位置**: `learning-path/`

🗺️ 为你定制系统化学习路径，从入门到精通！

**功能**:
- ✅ 生成系统学习路径
- ✅ 分阶段学习目标
- ✅ 推荐学习资源
- ✅ 进度追踪
- ✅ 时间规划建议
- ✅ 技能图谱可视化

**快速开始**:
```
学习路径 [技能/主题]
我的进度
今日任务
学习资源 [主题]
```

---

## 🚀 安装方法

1. 将技能文件夹复制到 OpenClaw skills 目录：
```bash
# 示例：复制 vocab-builder
cp -r vocab-builder ~/.openclaw/skills/
```

2. 在 OpenClaw 中启用技能：
```
启用技能 vocab-builder
```

3. 开始使用！

## 💡 使用建议

### 语言学习
组合使用：**Vocab Builder** + **Quiz Maker**
- 每天学习新单词
- 定期测验巩固

### 编程学习
组合使用：**Learning Path** + **Code Snippet** + **Quiz Maker**
- 按路径系统学习
- 保存常用代码
- 测验检验效果

### 阅读学习
组合使用：**Book Summary** + **Quiz Maker**
- 快速了解书籍核心
- 测验加深理解
