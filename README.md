## 📝 技能结构

每个技能文件夹包含：
- `SKILL.md` - 技能描述和触发规则
- `index.js` / `script.js` - 主要逻辑
- 其他辅助文件

## 🛡️ 安全提示

- 安装任何技能前请先运行 `skill-vetter` 审查
- 不要提交包含 API 密钥或敏感信息的文件
- 审查第三方技能的权限请求
