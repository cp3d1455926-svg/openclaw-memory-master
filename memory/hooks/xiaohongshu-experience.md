## 链接参数解析

小红书笔记链接包含多个安全参数：

```
https://www.xiaohongshu.com/discovery/item/6917ec65000000001b0201f7
?source=webshare              # 分享来源：网页分享
&xhsshare=pc_web              # 分享渠道：PC 网页
&xsec_token=ABXxxx            # 安全令牌（重要！）
&xsec_source=pc_share         # 安全来源
```

**关键参数**:
- `xsec_token`: 安全令牌，每次请求需要，会过期
- `xsec_source`: 来源标识（pc_share, app_share, copy_link 等）
- `source`: 分享来源（webshare, copy_link 等）

```

### 流程 2: 发布笔记

```
1. 点击"发布笔记"按钮

2. 上传图片/视频

   - 可加话题标签 #xxx#
   - 可@其他用户

   - 增加曝光度

6. 选择封面
   - 从上传的图片中选择
   - 或单独上传封面

7. 点击发布
   - 等待审核（通常几分钟）
   - 审核通过后公开可见

**注意事项**:
- 不要发布违规内容
- 避免敏感词汇
- 图片不要有水印
- 遵守社区规范

## 已知陷阱

### 陷阱 1: xsec_token 过期

**现象**: 
- 请求返回 403
- 重定向到登录页
- 内容无法加载

**原因**: xsec_token 已过期或无效

**解决**:
```
1. 重新打开笔记页面
2. 复制新的链接（包含新的 xsec_token）
3. 使用新链接访问
```

**预防**:
- 不要保存旧链接
- 每次使用都重新获取
- 或使用官方 API（如果有权限）

---

### 陷阱 2: 创作者平台状态校验

**现象**: 无法发布内容或功能受限

**原因**: 
- 账号未通过创作者认证
- 账号有违规记录
- 新账号限制

**解决**:
```
1. 完成实名认证
2. 申请创作者认证
3. 等待审核通过
4. 遵守平台规则
```

---

### 陷阱 3: 内容审核

**现象**: 发布后内容不可见

**原因**: 
- 内容正在审核中
- 内容违规被限制
- 账号被限流

**解决**:
```
1. 等待审核完成（通常几分钟到几小时）
2. 检查是否有违规内容
3. 修改后重新发布
4. 如被误判，联系官方客服
```

**预防**:
- 不要发布敏感内容
- 不要抄袭
- 不要频繁发布
- 使用小号测试

---

### 陷阱 4: 图片水印

**现象**: 下载的图片有小红书水印

**原因**: 平台自动添加水印

**解决**:
```
方法 1: 使用官方分享功能
  - 分享到其他平台会自动去水印
  
方法 2: 使用第三方工具
  - 有风险，不推荐
  
方法 3: 手动裁剪
  - 费时，但安全
```

---

### 陷阱 5: 登录态失效

**现象**: 
- Cookie 失效
- 需要重新登录
- 自动化脚本无法执行

**原因**: Cookie 过期

**解决**:
```
1. 重新登录
2. 获取新的 Cookie
3. 更新到自动化脚本
4. 设置定期刷新机制
```

**预防**:
- 使用"记住我"功能
- 定期检查登录态
- 准备备用账号

## 技术事实

### 事实 1: 链接参数机制

**xsec_token**:
- 每次访问生成新的 token
- 有效期短（几分钟到几小时）
- 与用户会话绑定
- 用于防爬和安全验证

**xsec_source**:
- 标识来源类型
- 常见值：pc_share, app_share, copy_link
- 用于统计和风控

**source**:
- 分享渠道
- 常见值：webshare, copy_link, qq, weixin
- 用于归因分析

---

### 事实 2: 登录态保持

**Cookie 关键信息**:
```
web_session: 会话 ID
gid: 用户 ID
gid.y: 用户 ID（备份）
xsec_token: 安全令牌（动态）
```

**有效期**:
- web_session: 约 7 天
- xsec_token: 几分钟到几小时
- 需要定期刷新

---

### 事实 3: 内容提取

**HTML 结构**:
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "笔记标题",
  "author": {
    "@type": "Person",
    "name": "作者名"
  },
  "datePublished": "2026-04-06",
  "image": ["图片 URL"],
  "articleBody": "笔记内容"
}
</script>
```

**提取方法**:
```javascript
// 使用浏览器控制台
const jsonLd = document.querySelector('script[type="application/ld+json"]');
const data = JSON.parse(jsonLd.textContent);
console.log(data);
```

**检测机制**:
