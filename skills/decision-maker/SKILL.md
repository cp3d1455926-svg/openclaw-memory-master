# Decision Maker - 选择困难症助手

🎲 帮你做决定，治愈选择困难症！

## 功能

- 🎲 随机选择器
- ⚖️ 利弊分析
- 📊 决策矩阵
- 🎯 权重评分
- 🔮 命运硬币
- 📋 选项对比
- 🎰 幸运转盘
- 📅 时间规划

## 使用方法

### 随机选择
```
帮我决定 吃火锅还是吃烤肉
选择 奶茶 咖啡 果汁
随机选择 选项 1 选项 2 选项 3
```

### 利弊分析
```
分析利弊 要不要换工作
决策分析 买不买房
```

### 权重评分
```
评分决策 工作选择 [薪资 8 分 距离 6 分 发展 9 分]
```

### 特殊工具
```
抛硬币 正面就去 反面不去
抽签 去 不去 再看看
转盘决定 选项 1 选项 2 选项 3
```

## 特性

✅ 多种决策方式
✅ 理性分析工具
✅ 随机但不随意
✅ 决策记录
✅ 结果可追溯
✅ 朋友一起决定
✅ 决策建议

## 配置

在 `config.json` 中设置：
- `defaultMode`: 默认模式
- `allowRandom`: 允许随机
- `saveHistory`: 保存历史
- `showAnalysis`: 显示分析
