# Pollinations.AI 图像生成助手

## 核心功能

### 1. 文生图
- 根据文字描述生成图片
- 支持多种艺术风格
- 免费、无需 API Key

### 2. 图生图
- 基于参考图生成新图片
- 保持风格一致性
- 支持多种变换

### 3. 多种风格
- 写实风格
- 动漫风格
- 油画风格
- 水彩风格
- 像素艺术
- 3D 渲染

### 4. 自定义尺寸
- 支持多种分辨率
- 最大 2048x2048
- 自定义宽高比

## API 说明

### 基础 URL
```
https://image.pollinations.ai/prompt/{prompt}
```

### 参数
| 参数 | 说明 | 示例 |
|------|------|------|
| prompt | 文字描述（必填） | "a cute cat" |
| width | 宽度（可选） | 1024 |
| height | 高度（可选） | 1024 |
| seed | 随机种子（可选） | 12345 |
| model | 模型（可选） | "flux" |

### 完整 URL 示例
```
https://image.pollinations.ai/prompt/a%20cute%20cat?width=1024&height=1024&seed=12345
```

## 提示词技巧

### 基础结构
```
[主体] + [动作/状态] + [环境] + [风格] + [质量词]
```

### 示例
```
一个科技感十足的 AI 助理，坐在电脑前工作，未来办公室背景，赛博朋克风格，高质量，细节丰富
```

### 质量词
- high quality（高质量）
- detailed（细节丰富）
- professional（专业）
- 8k resolution（8K 分辨率）
- masterpiece（杰作）

### 风格词
- cyberpunk（赛博朋克）
- realistic（写实）
- anime（动漫）
- oil painting（油画）
- watercolor（水彩）
- pixel art（像素艺术）
- 3D render（3D 渲染）

- 有速率限制（约 10 张/分钟）

- 生成的图片可商用

## 故障排除

### 问题 1：图片生成失败
```
检查网络连接
检查提示词是否包含敏感词
稍后再试（可能服务器繁忙）
```

### 问题 2：图片质量不好
```
添加质量词：high quality, detailed
增加尺寸：width=2048, height=2048
使用更好的模型：model=flux
```

### 问题 3：生成速度慢
```
降低图片尺寸
避开高峰期
使用本地缓存
```
