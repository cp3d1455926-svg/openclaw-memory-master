# Resume Builder - 简历生成器

📄 智能简历生成器，快速制作专业简历！

## 功能

- 📝 简历模板生成
- 🎨 多种样式选择
- ✨ 内容优化建议
- 📊 技能匹配分析
- 🔍 ATS 优化
- 📱 手机适配版本
- 🌐 中英文版本
- 📤 多格式导出

## 使用方法

### 创建简历
```
创建简历
生成简历 模板：简约
简历 职位：产品经理
```

### 优化简历
```
优化简历 [上传现有简历]
简历诊断 [简历内容]
```

### 定制版本
```
英文简历
应届生简历
转行简历
```

### 导出简历
```
导出简历 PDF
导出简历 Word
```

### 简历诊断
```
📊 简历诊断报告

📄 原简历评分：75/100

━━━ 优点 ━━━

✅ 结构清晰，层次分明
✅ 工作经历描述详细
✅ 技能列表完整

━━━ 需改进 ━━━

⚠️ 缺少量化数据（重要度：高）
   建议：添加具体数字和百分比

⚠️ 动词使用单一（重要度：中）
   建议：使用"主导""推动""实现"等强动词

⚠️ 缺少项目成果（重要度：高）
   建议：补充项目影响和业务价值

⚠️ 自我评价过于笼统（重要度：中）
   建议：突出个人特色和核心竞争力

━━━ ATS 优化建议 ━━━

• 添加职位关键词：产品规划、需求分析
• 使用标准职位名称
• 避免图片和表格
• 使用标准字体

📈 优化后预计评分：90/100
```

## 配置

在 `config.json` 中设置：
- `defaultTemplate`: 默认模板
- `outputFormat`: 输出格式
- `atsOptimize`: ATS 优化开关
- `language`: 默认语言
